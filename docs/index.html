★ ポップンメダル集計

2. ブックマークに登録

パソコン

  1. 上のリンクをブックマークバーにドラッグ。または「コピー」後、ブックマークを新規作成し、URL欄に貼り付け。

iPhone / iPad（Safari）

  1. 「コピー」を押す。
  2. このページで共有ボタン →「ブックマークを追加」で適当に保存。
  3. ブックマーク一覧 → いま追加したものを「編集」。
  4. URL欄を全部消して、コピーしたコードを貼り付け → 保存。

Android（Chrome）

  1. 「コピー」を押す。
  2. このページで ⋮ →「☆」でブックマーク保存。
  3. ブックマークを開いて鉛筆（編集）→ URLを貼り付けたコードに置き換え → 保存。
スマホでは、貼り付け後に URL が javascript: で始まっているか確認してください。先頭の javascript: が消えてしまう場合は、手入力で補ってください。

3. 使い方

  1. e-amusement にログインした状態で、pop'n のプレーデータのページ（p.eagate.573.jp/game/popn/popn29/playdata/index.html）を開く。
  2. 登録したブックマークをタップ／クリック。初回は全走査のため数分かかります。
  3. 2 回目以降はキャッシュが即表示されます。最新化は表の右下「更新」ボタンで。
e-amusement のベーシックコース登録済みアカウントが必要です。開発中のため、メダル数が正しくない場合があります。表示のみでデータの外部送信はありません。
